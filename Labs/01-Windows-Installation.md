#Lab 01 - Windows 10 Installation

## Objective

Install Windows 10 in Oracle VirtualBox to create the foundation for the IT Home Lab.

## Environment 

- Oracle VirtualBox
- Windows 10 Pro
- 4 GB RAM
- 2 CPU CORES
- 60 GB Virtual Hard disk

## Steps Performed

1. Created a new Virtual Machine
2. Attached The windows 10 ISO
3. Installed Windows 10 pro
4. Created a local Admin account
5. Completed The out of box experience 

## Why It Matters
Installing Windows is a foundational IT skill. Help desk and desktop support technicians deploy operating systems, replace failed drives, prepare new computers, and configure devices for end users.

## Skills Practiced
- Virtual machine creation
- Windows operating system deployment
- Local account creation
- Windows setup (OOBE)
