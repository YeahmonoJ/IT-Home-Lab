#Lab 01 - Windows 10 Installation

## Objective

Install Windows 10 in Oracle VirtualBox to create the foundation for the IT Home Lab.

## Environment 

- Oracle VirtualBox
- Windows 10 Pro
- 4 GB RAM
- 2 CPU CORES
- 60 GB Virtual Hard Disk

## Steps Performed

1. Created a new Virtual Machine
2. Attached The Windows 10 ISO
3. Installed Windows 10 pro
4. Created a local administrator account (LabAdmin).
5. Completed the Out-of-Box Experience (OOBE).
6. Installed VirtualBox Guest Additions
7. Restarted the virtual machine.

## Why It Matters
Installing Windows and VirtualBox Guest Additions prepares a virtual machine for daily administration. Guest Additions installs optimized drivers and improves usability by enabling better display support, seamless mouse integration, and clipboard sharing.

## Skills Practiced
- Virtual machine creation
- Windows operating system deployment
- Local account creation
- Windows setup (OOBE)
- VirtualBox Guest Additions installation
- Basic driver installation

- Windows setup (OOBE)
Successfully deployed a Windows 10 virtual machine that will be used for future networking, troubleshooting, and administration labs.
